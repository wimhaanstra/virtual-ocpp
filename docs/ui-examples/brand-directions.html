More distinct product directions

These move away from rounded card-heavy defaults: tighter spacing, smaller radii, stronger rails, and a more recognizable primary color. Each option shows a dark dashboard and a light Communication page with events merged in.

Ion teal #00A7A5 Circuit green #77B900 Charge amber #FF9D00 Protocol blue #2F6BFF

Direction 1: Ion Teal

Product-like without looking like a component library default. Teal fits energy and connectivity, works well in both themes, and keeps warning/error colors clear.
