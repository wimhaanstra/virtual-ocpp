Virtual OCPP UI direction examples

Each direction shows the same target UX: no Activity page, operational events folded into Communication, compact filters, icon-only actions, flatter surfaces, cleaner dashboard, and sectioned modal forms.

Option A: Crisp Ops UI

Neutral slate surfaces with blue/cyan accents. This is the recommended direction: dense enough for troubleshooting, calm enough for daily admin use, and easy to support in both dark and light modes.
